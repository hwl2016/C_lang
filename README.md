# C_lang
> C、C++、算法、数据结构学习

### C语言环境搭建、程序运行
1. vscode安装C/C++扩展
2. Windows安装mingw  Mac自带gcc/g++
3. vscode编辑完C语言代码后 ```右击 → Run Code```

### mingw安装gcc/g++/
```
mingw-get install gcc
mingw-get install g++
```

#### gcc/g++ 使用方法

##### 编译C语言文件
> gcc xxx -o xxx.exe

##### 编译C++语言
> g++ xxx -o xxx.exe
